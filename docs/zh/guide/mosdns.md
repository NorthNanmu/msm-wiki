# MosDNS 管理

MosDNS 负责 DNS 分流、缓存与规则匹配，是 MSM 的必选组件。

## 服务控制

- 启动 / 停止 / 重启
- 查看运行状态与版本
- 查看启动日志与错误提示

## 配置管理

1. 进入“配置编辑”页面选择 MosDNS 配置
2. 在线编辑并保存
3. 使用“配置校验”检查语法
4. 提示重启时执行重启以生效

## 常见配置项

- 上游 DNS
- 分流规则（国内 / 国外 / 广告）
- 缓存策略
- 日志级别

## 与 FakeIP 协同

如果启用 FakeIP 分流，请确保：

- MosDNS 的 FakeIP 段与 Mihomo / Sing-box 配置一致
- 主路由配置了 FakeIP 与关键 IP 的静态路由

详情参考“基础配置”页面中的主路由路由规则。
