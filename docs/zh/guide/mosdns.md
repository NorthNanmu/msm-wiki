# MosDNS 管理

MosDNS 是 MSM 的核心 DNS 服务，负责分流、缓存与日志。

## 页面入口

- **概述**：运行状态、版本、资源概览
- **规则管理**：域名规则与策略维护
- **客户端设置**：白名单 / 黑名单 / 关闭模式
- **DNS 日志**：查询记录与筛选
- **系统功能**：高级开关与系统参数
- **配置管理**：MosDNS 配置文件管理
- **实时日志**：启动与运行日志

## 常用操作

1. **修改分流规则**
   - 进入 **规则管理**，按域名/规则类型进行调整
2. **设置设备名单**
   - 进入 **客户端设置**，选择白/黑名单并添加设备
3. **排查 DNS 问题**
   - 进入 **DNS 日志** 查看具体请求与命中规则
   - 进入 **实时日志** 查看运行错误

## 与代理核心的协同

- FakeIP 模式需确保 **主路由**配置了 `28.0.0.0/8` 的静态路由
- 代理核心（Sing-box/Mihomo）与 MosDNS 的分流策略应保持一致
