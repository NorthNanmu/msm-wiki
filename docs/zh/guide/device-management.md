# 设备管理（客户端设置）

MSM 通过 MosDNS 的 **客户端设置** 管理设备的代理权限，支持 **关闭 / 白名单 / 黑名单** 三种模式。

## 功能说明

- **关闭**：不启用名单控制
- **白名单**：仅白名单内设备走代理
- **黑名单**：黑名单内设备不走代理，其他设备走代理

> 名单由 MosDNS 的 `client_ip` 规则生效，模式切换会决定名单是白名单还是黑名单。

## 推荐流程

1. 进入 **MosDNS > 客户端设置**
2. 选择代理模式（关闭 / 白名单 / 黑名单）
3. 点击 **扫描局域网** 获取在线设备
4. 将设备拖入白名单或黑名单（或使用按钮添加/移除）
5. 保存后生效（必要时按页面提示重启服务）

## 常见操作

- **添加设备**：点击“添加”或从扫描结果拖拽到名单
- **批量操作**：勾选多台设备进行批量删除
- **切换模式**：在页面顶部切换“关闭 / 白名单 / 黑名单”
- **查看设备信息**：点击设备卡片查看 IP / MAC / 主机名

## 注意事项

- 如果启用了白/黑名单，**未被允许的设备**可能无法访问需要代理的域名
- 修改名单后，请观察 **MosDNS > 实时日志** 与 **DNS 日志** 是否生效
