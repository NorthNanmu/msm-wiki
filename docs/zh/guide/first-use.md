# 首次使用

本页整理 MSM 首次初始化的关键步骤。

## 1. 访问系统

打开浏览器访问：

```
http://your-server-ip:7777
```

## 2. 初始化向导

首次访问会进入初始化流程，主要包含：

1. **管理员账号**：设置管理员用户名与密码
2. **服务选择**：
   - MosDNS 必选
   - Sing-box 与 Mihomo 二选一（不可同时启用）
3. **组件下载**：系统自动下载并解压所选组件
4. **基础配置**：初始化 DNS 与代理的默认配置

## 3. 登录与仪表板

完成初始化后使用管理员账号登录，即可进入仪表板查看：

- 三大服务运行状态
- 资源使用情况（CPU / 内存 / 磁盘 / 网络）
- 最近操作与告警信息

## 4. 下一步建议

- 完成路由与 DNS 基础配置（见“基础配置”）
- 编辑 MosDNS 与代理配置并重启服务
- 建立配置历史与备份策略
