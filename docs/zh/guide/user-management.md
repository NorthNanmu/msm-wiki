# 用户管理

MSM 提供多用户登录与权限控制，适合多人协作与运维分工。

## 核心能力

- 创建 / 编辑 / 禁用用户
- 角色与权限控制（只读 / 运维 / 管理员等）
- 登录日志与操作审计

## 常见操作

### 新增用户

1. 进入“用户管理”页面
2. 点击“新增用户”
3. 填写用户名、密码与角色
4. 保存后即可登录使用

### 修改密码

- 管理员可在用户详情中重置密码
- 用户本人可在“个人设置”中修改密码

### 禁用用户

- 对离职或暂停权限的账号进行禁用
- 禁用后无法登录，但历史记录保留

## 安全建议

- 管理员账号避免共享
- 使用强密码并定期更换
- 开启日志审计，定期检查异常登录
