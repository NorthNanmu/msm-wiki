# 爱快（iKuai）配置指南

## 示例环境

- 爱快网关：`192.168.1.1`
- MSM 主机：`192.168.1.2`

## 步骤一：添加静态路由（FakeIP）

在 **IPv4 路由/静态路由** 页面新增一条路由：

- **目的 IP 地址**：`28.0.0.0`
- **掩码或前缀长度**：`255.0.0.0` 或 `/8`
- **下一跳 IP**：`192.168.1.2`
- **距离**：保持默认即可

## 步骤二：配置 DHCP DNS

在 **DHCP 服务器地址池** 中设置 DNS：

- **DNS 服务器**：`192.168.1.2`
- **备用 DNS**：可选填写运营商 DNS

## 验证

客户端执行：

```bash
nslookup google.com
```

应返回 `28.0.0.0/8` 段地址。
