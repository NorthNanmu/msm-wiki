# Mihomo 管理

Mihomo 是 MSM 支持的代理内核之一，与 Sing-box 二选一。

## 页面入口

- **概览**：核心状态与关键配置概览
- **代理节点**：节点与策略组管理
- **规则管理**：规则与策略调整
- **连接管理**：实时连接与流量统计
- **配置管理**：编辑配置文件并校验
- **日志查看**：运行日志与错误排查

## 推荐流程

1. 在初始化向导选择 **Mihomo** 作为代理核心
2. 进入 **代理节点** 导入订阅或手动配置节点
3. 在 **规则管理** 调整策略规则
4. 通过 **连接管理** 观察实际流量走向

## 注意事项

- FakeIP 网段需与 MosDNS 配置保持一致
- 修改配置后需重启服务生效
