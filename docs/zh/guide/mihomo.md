# Mihomo 管理

Mihomo 是 MSM 支持的代理内核之一，与 Sing-box 二选一。

## 启用与切换

- 初始化阶段选择 Mihomo 后，系统会自动下载与配置
- 如需切换内核，请先停止当前代理服务，再执行切换操作

## 服务控制

- 启动 / 停止 / 重启
- 查看运行状态与版本
- 查看启动日志

## 配置管理

- 在线编辑 Mihomo 配置
- 保存后重启生效
- 建议先通过配置校验再重启

## FakeIP 与透明代理

- FakeIP 网段需与 MosDNS 保持一致
- 若启用透明代理，确保主路由与防火墙规则已配置
- 变更后观察日志与流量是否正确分流
