# 前端开发

## 技术栈

- React 19 + Vite
- React Router v7
- Zustand
- TailwindCSS 4
- CodeMirror（配置编辑）
- i18next（国际化）

## 本地开发

```bash
cd frontend
npm install
npm run dev
```

## 目录约定

- `src/pages`：页面入口
- `src/components`：通用组件
- `src/lib`：API 与工具封装
- `src/stores`：全局状态

## 测试

```bash
npm test
npm test -- --run
```
