# 项目结构

MSM 项目主要目录如下：

```
msm/
├── backend/        # Go 后端服务
├── frontend/       # React 前端
├── docs/           # 设计与运维文档
├── build/          # 构建与打包脚本
├── network/        # 网络与代理相关配置
└── logo/           # 品牌资源
```

## 后端目录要点

- `backend/cmd`：命令入口
- `backend/internal`：业务逻辑与 API
- `backend/configs`：配置模板与示例

## 前端目录要点

- `frontend/src/pages`：页面
- `frontend/src/components`：组件
- `frontend/src/lib`：API 与工具
- `frontend/src/stores`：状态管理
