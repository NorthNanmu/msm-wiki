# What is MSM

MSM (MosDNS Sing-box Mihomo Manager) is a unified Web platform to manage MosDNS, Sing-box, and Mihomo.

## Key Goals

- One UI for all core services
- Visual configuration editing and history rollback
- Real-time status, logs, and system monitoring
- Role-based access control

## Supported Services

- MosDNS (DNS split & cache)
- Sing-box (proxy core)
- Mihomo (proxy core)
