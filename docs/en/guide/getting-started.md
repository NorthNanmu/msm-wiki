# Getting Started

## Quick Install

```bash
curl -fsSL https://raw.githubusercontent.com/msm9527/msm-wiki/main/install.sh | sudo bash
```

After installation, visit:

```
http://your-server-ip:7777
```

## Manual Run

```bash
msm
msm -d
msm -p 8080
```
